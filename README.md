# 🛒 Proyecto Clucky - Tienda de Ropa Online

Bienvenido al repositorio de **Clucky**, una aplicación web de ventas de ropa desarrollada con Laravel.

## 🚀 Requisitos del Sistema
Para poder ejecutar este proyecto, asegúrate de tener instalado:
- **PHP 8.2+** (Incluido en XAMPP)
- **Composer** (Gestor de dependencias de PHP)
- **Node.js & NPM** (Para compilar los estilos de la interfaz)
- **MySQL** (Servidor de base de datos)
